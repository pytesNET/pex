PEX - Printer Execution eXchange
================================

Printer execution exchange server, service and utilities for cross-os compatibility.

## Installation

1. Create a virtual environment:

```sh
python -m venv .venv
```

2. Activate virtual environment (Linux)

```sh
source .venv\bin\activate
```

2. Activate virtual environment (Windows)

```sh
.venv\Scripts\activate
```

3. Install requirements

```sh 
pip install -r requirements.txt
```
